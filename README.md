#### 📷 [LensMaster Pro](https://github.com/AlAleksandrov/lensmaster_pro)

A **production-ready Django 6.0 web application** for photography & videography studios.

- 🏗️ Multi-app architecture (Bookings / Productions / Inventory)
- 🎯 Full CRUD with server-side validation
- 🗄️ PostgreSQL with environment-based config
- 🎨 Bootstrap 5 dark theme with custom design
- 📄 Pagination, slug auto-generation, custom 404
- 🔧 Auto-generated inventory IDs (`INV-XXXX`)
